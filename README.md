# 🌐 QA Project – Frontend (React)

This repository contains the **frontend** of my second-year QA project, built with **React**. It works with a separate backend repository and includes **Selenium tests** for automated UI testing.

[![React](https://img.shields.io/badge/React-17.0.2-blue?logo=react&logoColor=white)](https://reactjs.org/)  
[![Selenium](https://img.shields.io/badge/Selenium-4.12.0-green?logo=selenium&logoColor=white)](https://www.selenium.dev/)  

---

## 📝 Project Overview

- Frontend for a QA project using **React**  
- Supports connection to a separate backend repository  
- Includes automated **Selenium UI tests**  
- Part of a **dual-repo CI/CD pipeline setup**  

---

## ⚡ Features

- Interactive UI for the QA project  
- Dynamic data fetching from backend API  
- Automated Selenium tests for core workflows  
- Fully responsive design
